MULAI DARI SINI - AERIZEN BLANK SCREEN FIX 1.0.2

Perbaikan utama:
1. Electron sekarang mencari dist/index.html dari beberapa lokasi.
2. Menu default File/Edit/View disembunyikan.
3. Jika UI gagal, aplikasi akan menampilkan pesan error, bukan layar kosong.
4. Packaging memakai asar=false agar file UI lebih mudah dibaca oleh Electron.

Langkah wajib setelah build:
1. Uninstall Aerizen Asset Management versi lama dari Windows.
2. Restart laptop/PC.
3. Install Aerizen-Asset-Management-Setup-1.0.2.exe.
4. Jangan jalankan shortcut lama dari Start Menu sebelum uninstall.

Kalau masih kosong dan masih ada menu File/Edit/View, berarti yang dibuka masih installer/app lama, bukan versi 1.0.2 ini.
